# Typora 使用教程

# 常用快捷键

  • 加粗：ctrl+B （bold）
  • 字体倾斜：ctrl+I （incline）
  • 下划线：ctrl+U （under）
  • 多级标题：ctrl+1~6
  • 有序列表：ctrl+shift+[
  • 无序列表：ctrl+shift+]
  • 降级：Tab
  • 升级：shift+Tab
  • 插入链接：ctrl+K （link）
  • 插入公式：ctrl+shift+M
  • 代码：ctrl+shift+K
  • 插入图片：ctrl+shift+I （image）
  • 创建表格：ctrl+T （table）
  • 选中一行：ctrl+L （line）
  • 选中一字：ctrl+D
  • 选中相同格式的文字：ctrl+E
  • 搜索：ctrl+F
  • 搜索并替换：ctrl+H
  • 删除线：Alt+shift+5
  • 引用：ctrl+shift+Q
  • 生成目录：[TOC]+Enter
  • 分割线：---+Enter

# 字体

  1. 斜体：

    *斜体字体1*，_斜体字体2_

    斜体字体 1，斜体字体 2

  2. 加粗：

    **加粗字体1**，__加粗字体2__

    加粗字体 1，加粗字体 2

  3. 删除线：

    ~~需要删除的文字~~

    需要删除的文字

  4. 大高亮

    ==高亮==

    高亮

  5. 小高亮

    `高亮`

    高亮

# 段落

  1. 更小行间距 ：shift+Enter
  2. 打断线：
    打断线后的内容将自动换行
  3. 引用：>+Space + 文字

# 特殊符号

  1. < ：＆ｌｔ；

  2. ＆ ：＆ａｍｐ；

  • - + 空格 + [ + 空格 + ] + 空格

  1. 省略号

    • 低端对其的省略号（…\ldots…）： $\ldots$
    • 中线对其的省略号（⋯\cdots⋯）： $\cdots$
    • 竖直对齐的省略号（⋮\vdots⋮）： $\vdots$
    • 写对期齐的省略号（⋱\ddots⋱）： $\ddots$

# 跳转

  1. 文档之间跳转

    • 相对路径（适用于 markdown 的阅读）：

      [content](相对地址)

      如果在相对文件夹下，相对地址为 ./xxx.md ;

      ../ 用于表示上一级目录，以此类推；

    • 绝对路径（适用于博客的阅读）：

      HTML 写法：

      Markdown 写法：

      [content](xxx.com)  //快捷键 ctrl+

  2. 文档内跳转

    第一步：定义一个锚 (id)

    第二步：

    [点击跳转](#jump)
